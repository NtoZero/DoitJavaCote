# Java Coding Test 
코딩 테스트를 위한 스터디입니다.
자세한 내용은 [ABOUT STUDY](https://github.com/NtoZero/DoitJavaCote/wiki) 에서 확인가능합니다.


## 스터디 목적
- 코딩테스트에 대한 감을 익힌다.
- 다양한 알고리즘에 대한 이해와 자료구조 이해
- 서로의 풀이법을 공유하며 문제 풀이에 대해 다양한 시각을 가진다.
- 다른 팀원에게 공유하고 설명하며 커뮤니케이션 능력을 기른다.
